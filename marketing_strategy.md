# Grand Line Cards - Marketing Strategy

## Mission
To become the #1 Spanish-speaking source for One Piece Card Game data, news, and trading.

## Target Audience
1. **Competitive Players**: Need meta analysis, deck lists, card database.
2. **Collectors**: Need price tracking, "Grand Line Vault" secure storage, card variants.
3. **New Players**: Need "Pirate Academy" (how to play), starter guides.

## Channels & Tactics

### 1. SEO & Content Marketing (Organic)
- **Strategy**: Dominate keywords like "One Piece Card Game España", "OP-05 Card List", "Luffy Leader deck".
- **Actions**:
    -   Maintain `/cartas` (Encyclopedia) with extensive metadata/SEO tags.
    -   Publish weekly Meta Tier Lists (`/meta`).
    -   Blog posts for set reviews and patch notes.

### 2. Community Building
- **Discord**: Official GLC server for LFG, trades, and feedback.
- **Local Game Stores (LGS)**: Partner with Spanish stores for "Grand Line Circuit" tournaments.
- **Twitter/X**: Auto-post new cards, price spikes (using Price Intelligence Engine).

### 3. "Grand Line Vault" Launch
- **Value Prop**: Save on shipping. Consolidate purchases.
- **Launch Comp**: "First 100 users get free Vault storage for 1 year."

### 4. Pirate Academy (Acquisition)
- **Funnel**: Traffic -> `/academia` -> Read Rules -> Buy Starter Deck (Affiliate/Direct) -> Register.

## KPI Goals (Q1)
- **Registered Users**: 10,000.
- **Daily Active Users**: 1,500.
- **Marketplace GMV**: €50,000.
- **SEO Traffic**: 50,000 monthly visits.
